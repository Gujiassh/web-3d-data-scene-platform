# Factory Assets

The M0 asset is a deterministic contract fixture generated by Node.js. It is
not the final Blender-authored Factory Demo asset set planned for M3.

```bash
node assets/factory/scripts/generate-m0-asset.mjs
```

Generated GLB files in `public/` are released under CC0-1.0. Generator source
uses the repository's MIT license.

The generator also writes `m0-scene.json` and a node manifest. All M0 packages
and apps use these files as the shared contract fixture.
